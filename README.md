# Snake game

Classic snake game written in pure python with zero external dependencies.
Game is complete. Certain refactoring has to be done.

## Requirements

- Python 3.7.2
    - The game was developed in 3.7.2.
    
## Running the game

- Open terminal/cmd
- Navigate to this repo.
- Run `python snake.py`
